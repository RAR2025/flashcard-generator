
# Flashcard Generator (Fullstack)

## Setup

### Backend
```
cd server
npm install
node server.js
```

### Frontend
```
cd client
npm install
npm start
```

## Environment
- Set OPENAI_API_KEY in server/.env
- Set REACT_APP_API_URL in client/.env
